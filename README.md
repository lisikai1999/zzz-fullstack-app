# ZZZ Fullstack App

A full-stack web application with frontend and backend.

## Project Structure

```
zzz-fullstack-app/
├── frontend/     # React + Vite frontend
├── backend/      # Express.js backend
└── README.md
```

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
